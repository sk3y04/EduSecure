# Feature Specifications

Feature documentation is grouped by functionality area.

## Subfolders

- `authentication/` — auth contract, MFA details, and auth-phase planning notes
- `submissions-audit/` — secure submission, signatures, storage, and audit design
- `grades/` — grade API and grade-integrity scope rules
- `academic-workflows/` — spaces, per-space chat, registration requests, exams, feedback, and attendance specs
- `frontend/` — frontend MVP and integration planning
