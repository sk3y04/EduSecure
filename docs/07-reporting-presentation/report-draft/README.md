# Report Draft Workspace

This directory is a working area for new report-writing material before it is merged into the larger final draft files.

## Current drafting approach

1. establish the report methodology wording
2. prepare short report-ready paragraphs for key sections
3. add comparison tables where they improve clarity
4. expand each section in the order recommended by `high-mark-report-blueprint.md`
5. merge refined text into the final report draft once evidence links and wording boundaries are checked
